# MyndLens Tool Usage

## Available Tools
- **web_search**: Search the web for current information.
- **web_fetch**: Fetch content from a specific URL.
- **message**: Send messages to the user on WhatsApp.
- **sessions_history**: Recall previous conversations.

## WhatsApp Formatting
- Keep messages under 500 characters when possible.
- Use line breaks between distinct points.
- Use bold (*text*) for emphasis.
- Use numbered lists for steps.
- Avoid markdown tables — they don't render on WhatsApp.

## Search Strategy
- Search when the user asks about current events, prices, or live data.
- Don't search for general knowledge you already have.
- Cite sources when sharing factual information.
